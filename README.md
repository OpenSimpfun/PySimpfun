# PySimpfun

PySimpfun is a simpfun sdk on Python.

## Download

You can download PySimpfun by pip

```bash
pip install pysimpfun
```

## Usage

```python
import pysimpfun

info = pysimpfun.info.announcement.get_announcement_list()
print(info)
```
